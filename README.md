# AnalisisPropios

Aquí van cuatro análisis propios para tener un portfolio.
- El primero se llama SocialMedia
que va sobre, en un rango de edad de 18 a 24 años, el impacto que ocasionan las rrss en el rendimiento académico. 

- El segundo se llama Ps4
que es, un histórico de ventas de videojuegos de ps4, ya que colecciono videojuegos y es uno de mis hobbies favoritos.

- El tercero se llama GitHub
que va sobre el uso propio de GitHub, en qué lenguaje se usa mas entre otras cosas.

- El cuarto se llama Café
que por último, aunque no peor, va sobre las ventas de café, tipos de café, modos de trasaccióm y demás.

En todos hay todo tipo de notas, gráficos, librerias como matplotlib o seaborn, scatters, boxplots, predicciones, clasificaciones como RandomForest. tambien se usa numpy para cálculos numéricos, entre otras cosas. Se identifican las variables, y mucho más. Todo va acompañado con su csv. No descartes, que haya algún tipo de error
Recuerda que soy principiante y estoy aprendiendo, pero no me avergüenzo de nada!

## Cómo ejecutar los análisis

Empieza clonando este repositorio:

   ```bash
   git clone https://github.com/NoeliaTJ/AnalisisPropios.git
   cd AnalisisPropios
   ```

Crea y activa un entorno virtual (opcional pero recomendado):

   ```bash
   python -m venv venv
   source venv/bin/activate  # si eres de macOS / Linux como es mi caso
   # .\venv\Scripts\activate  # si eres de Windows
   ```

Instala las dependencias (si existe un archivo `requirements.txt`):

   ```bash
   pip install -r requirements.txt
   ```

4. Entra en la carpeta del análisis que quieras revisar (por ejemplo, `SocialMedia`, `Ps4`, `GitHub` o `Cafe`) y abre el notebook o script que mas te guste.
